Sistema RFID para Carrinhos de Choque

Demonstração do Protótipo

Descrição do Projeto

Desenvolvimento de um sistema de autenticação baseado em RFID para carrinhos de choque, visando combater a falsificação de fichas produzidas em impressoras 3D caseiras. O sistema utiliza um leitor RFID e um Arduino para validar as fichas e controlar a alimentação dos carrinhos.

Funcionalidades Principais

  • Leitura de tags RFID
  • Validação de fichas autorizadas
  • Controle de alimentação via relé
  • Interface de administração
  • Registro de uso

Tecnologias Utilizadas

Arduino RFID Python Eletrônica

Desafios e Soluções

O principal desafio do projeto foi a viabilidade económica da produção das fichas com chip RFID. O investimento inicial em moldes para produção em massa representava um custo significativo, tornando o projeto inviável para uma única empresa de carrinhos de choque. A solução encontrada foi desenvolver um sistema modular que pode ser facilmente adaptado para diferentes clientes, permitindo diluir o custo de produção das fichas entre múltiplos utilizadores. Desta forma, o investimento torna-se viável a partir do segundo cliente, criando um modelo de negócio sustentável para a implementação do sistema.
