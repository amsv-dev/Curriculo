Sistema de Triagem Médica Inteligente

Descrição do Projeto

O Sistema de Triagem Médica Inteligente é uma aplicação web desenvolvida para auxiliar profissionais de saúde no processo de triagem médica. Utilizando o Teorema de Bayes, o sistema calcula probabilidades de diferentes condições médicas com base nos sintomas apresentados pelos pacientes.

Funcionalidades Principais

  • Interface intuitiva para entrada de sintomas
  • Cálculo de probabilidades usando Teorema de Bayes
  • Sistema de autenticação para médicos e pacientes
  • Histórico de triagens por paciente
  • Exportação de relatórios em PDF

Tecnologias Utilizadas

React Flask Python MySQL Bootstrap

Desafios e Soluções

Um dos principais desafios foi implementar o algoritmo de Bayes de forma eficiente, considerando a complexidade dos cálculos de probabilidade. A solução envolveu a criação de uma estrutura de dados otimizada e a implementação de cache para resultados frequentes.
